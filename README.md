# Custom Audio Server for Create Tram Additons
An unoffical fork of the offical audio server for the Create Tram Additions.
It is based on node.js (the orginal was python) and is 3 times more efficent than the old one, it is self hostable or avaiable at backend.notjulian.me.
## Features
- Custom Sounds
- Efficent
- Self-hostable
# Warning
The IBIS Dashboard is with this server not compatible! A patch is currently in work.
